# Atm-Machine

Sistem ATM dengan sistem save menggunakan C++ pada console

## Usage

Linux
```sh
# Build
/path/to/g++ -g ./main.cpp ./includes/Atm.cpp -o ./main

# Running
./main
```

Windows
```sh
# Build Use g++
g++ ./main.cpp ./includes/Atm.cpp -o main.exe

# Build Use gcc
gcc ./main.cpp ./includes/Atm.cpp -o main.exe

# Running
./main.exe
```
